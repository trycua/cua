Reference documents the full technical specification for Cua's subsystems. It covers `cua-driver`, Lume, and `sandbox-sdk`. The pages define commands, APIs, exposed tools, types, and stated limits.

| Section | What it documents |
|---------|-------------------|
| [`cua-driver`](/reference/cua-driver/cli-reference) | CLI commands, 38 MCP tools exposed via the stdio MCP server, and known behavioral limits, including platform-specific constraints on the no-foreground-steal contract. |
| [Lume](/reference/lume/cli-reference) | CLI commands and local HTTP API for creating, running, and managing macOS and Linux VMs on Apple Silicon Macs. |
| [`sandbox-sdk`](/reference/sandbox-sdk) | Sandbox SDK API, documented by a separate agent. |
