# Cyclops TypeScript UniFFI bindings

Generated with `uniffi-bindgen-react-native` for Node.js N-API. Runtime
requires `@ubjs/core`, `@ubjs/node`, and a colocated `libcyclops_sdk` cdylib.
Browser source is generated in `../ts-uniffi-browser`; packaging its WASM crate
requires a project `ubrn.config.yaml` and a wasm build pipeline.
