# Installing Elasticsearch via DEB Package on Ubuntu/Debian

## Key Installation Steps

**Step 1: Import the PGP Key**
Download and install Elasticsearch's signing key using:
```sh
wget -qO - https://artifacts.elastic.co/GPG-KEY-elasticsearch | sudo gpg --dearmor -o /usr/share/keyrings/elasticsearch-keyring.gpg
```

**Step 2: Configure APT Repository**
First, ensure you have the transport package: `sudo apt-get install apt-transport-https`

Then add the repository:
```sh
echo "deb [signed-by=/usr/share/keyrings/elasticsearch-keyring.gpg] https://artifacts.elastic.co/packages/9.x/apt stable main" | sudo tee /etc/apt/sources.list.d/elastic-9.x.list
```

**Step 3: Install the Package**
```sh
sudo apt-get update && sudo apt-get install elasticsearch
```

For a specific version, use: `sudo apt-get install elasticsearch=<VERSION>`

**Step 4: Enable and Start the Service**
```sh
sudo systemctl daemon-reload
sudo systemctl enable elasticsearch.service
sudo systemctl start elasticsearch.service
```

**Step 5: Set the Elastic Password**
```sh
/usr/share/elasticsearch/bin/elasticsearch-reset-password -u elastic
```

**Step 6: Verify Installation**
Test connectivity using: `curl --cacert /etc/elasticsearch/certs/http_ca.crt -u elastic:$ELASTIC_PASSWORD https://localhost:9200`

The documentation also offers manual installation via downloading the `.deb` file directly if preferred.
