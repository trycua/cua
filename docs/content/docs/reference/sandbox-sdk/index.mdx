Install `cua-sandbox` and import it as `cua_sandbox` (often aliased to `cua`).

```bash
pip install cua-sandbox
```

## Lifecycle and service access

Use `Sandbox.create()` to provision, `connect()` to attach, and `destroy()` to delete.

#### `Sandbox.create`

```python
async def create(image: Image, *, name: Optional[str] = None, api_key: Optional[str] = None, local: bool = False, runtime: Optional['Runtime'] = None, cpu: Optional[int] = None, memory_mb: Optional[int] = None, disk_gb: Optional[int] = None, region: str = 'us-east-1', time_to_start: Optional[float] = None, request_timeout: Optional[float] = None, telemetry_enabled: bool = True) -> 'Sandbox'
```

Provision a new persistent sandbox and return it connected.

#### `Sandbox.connect`

```python
def connect(name: str, *, api_key: Optional[str] = None, local: bool = False, ws_url: Optional[str] = None, http_url: Optional[str] = None, container_name: Optional[str] = None, cpu: Optional[int] = None, memory_mb: Optional[int] = None, disk_gb: Optional[int] = None, region: str = 'us-east-1', telemetry_enabled: bool = True) -> Sandbox
```

Connect to an existing sandbox by name.

Supports both `await` and `async with`; leaving the context calls `disconnect()` without destroying the sandbox.

#### `Sandbox.ephemeral`

```python
async def ephemeral(image: Image, *, name: Optional[str] = None, api_key: Optional[str] = None, local: bool = False, runtime: Optional['Runtime'] = None, cpu: Optional[int] = None, memory_mb: Optional[int] = None, disk_gb: Optional[int] = None, region: str = 'us-east-1', time_to_start: Optional[float] = None, request_timeout: Optional[float] = None, telemetry_enabled: bool = True) -> Sandbox
```

Create an ephemeral sandbox that is automatically destroyed on exit.

#### `Sandbox.disconnect`

```python
async def disconnect() -> None
```

Drop the transport connection. The sandbox keeps running.

#### `Sandbox.destroy`

```python
async def destroy() -> None
```

Disconnect and permanently delete the sandbox (VM/container).

#### `Sandbox.list`

```python
async def list(*, local: bool = False, api_key: Optional[str] = None) -> 'list[SandboxInfo]'
```

List running and suspended sandboxes.

#### `Sandbox.get_info`

```python
async def get_info(name: str, *, local: bool = False, api_key: Optional[str] = None) -> 'SandboxInfo'
```

Get metadata for a specific sandbox.

Raises: `ValueError` when a requested local sandbox is not found.

#### `Sandbox.suspend`

```python
async def suspend(name: str, *, local: bool = False, api_key: Optional[str] = None) -> None
```

Suspend a running sandbox (save state).

#### `Sandbox.resume`

```python
async def resume(name: str, *, local: bool = False, api_key: Optional[str] = None) -> 'Sandbox'
```

Resume a suspended sandbox and return a connected Sandbox.

Raises: `ValueError` when local state is missing or has an unknown runtime.

#### `Sandbox.restart`

```python
async def restart(name: str, *, local: bool = False, api_key: Optional[str] = None) -> 'Sandbox'
```

Restart a sandbox (suspend then resume) and return a connected Sandbox.

#### `Sandbox.delete`

```python
async def delete(name: str, *, local: bool = False, api_key: Optional[str] = None) -> None
```

Permanently delete a sandbox.

Raises: `ValueError` when a requested local sandbox is not found.
