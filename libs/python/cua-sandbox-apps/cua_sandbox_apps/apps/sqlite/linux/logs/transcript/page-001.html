Three options are available:

### Option A: Amalgamation (Recommended for compilation)
```
sqlite-amalgamation-3530000.zip (2.80 MiB)
```

### Option B: Autoconf (Unix-friendly)
```
sqlite-autoconf-3530000.tar.gz (3.12 MiB)
```

### Option C: Complete Source Tree
```
sqlite-src-3530000.zip (13.84 MiB)
```

- **Debian/Ubuntu**: `sudo apt-get install sqlite3`
- **Fedora/RHEL**: `sudo dnf install sqlite`
- **Arch**: `sudo pacman -S sqlite`

For development versions:
```
https://sqlite.org/src (Dallas)
```
