# Installing IntelliJ IDEA on Linux

## System Requirements

IntelliJ IDEA on Linux requires:

**Hardware:**
- CPU with x86_64 or arm64 architecture and 4 cores
- 8 GB total RAM (3 GB available for IDE processes)
- 10 GB disk space
- 1280 x 720 display resolution

**Operating System:**
Supported distributions include "Ubuntu 22.04, 24.04 LTS," Fedora, Debian 13, and Amazon Linux 2023. The system needs glibc 2.28 or later. BSD-based systems are not supported.

## Installation Methods

**1. Toolbox App (Recommended)**
Download the tarball, unpack it, then execute `./bin/jetbrains-toolbox`. The application requires packages like `libxi6`, `libxrender1`, and `libgtk-3-bin`.

**2. Standalone Installation**
Download the .tar.gz file and extract it using: `sudo tar -xzf idea-*.tar.gz -C /opt`. Then run `./bin/idea.sh` from the extracted directory.

**3. Snap Package**
Install via: `sudo snap install intellij-idea --classic`. Note that snap installations "may encounter issues including performance degradation" and file management delays, so the Toolbox App is preferred.
