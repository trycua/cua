### Highlights

- **Computer server bind default** — `computer_server` now binds to `127.0.0.1`
  by default. Use `--host 0.0.0.0` when the server needs to accept external
  connections.
- **N1 browser-use agent loop** — Added the Yutori N1 browser-use agent loop as a native loop, with adapter and loop registration integrated into the agent stack.
- **GPT-5.4 native agent loop support** — Added GPT-5.4 support for the Computer tool, enabling native agent loop execution.
- **Interactive & non-interactive shell on sandboxes** — New `cua sb shell` and `cua sb exec` commands for interactive and non-interactive sandbox access.
