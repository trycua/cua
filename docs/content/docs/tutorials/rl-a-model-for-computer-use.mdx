In this tutorial you set up a reinforcement learning loop that trains a model on computer-use tasks, using sandboxes as parallel environments.

## What you will build

A training loop where a model acts in sandboxes, is scored on task success, and is updated from those rewards.

## Prerequisites

The Cua SDK installed, a Cua API key, and a training setup for the model you want to RL.

## Steps

1. Define the task environment and reward.
2. Run many sandboxes in parallel as rollout environments.
3. Score trajectories and update the model.
