title: "Use the SDK"
description: "Install the Cua Sandbox Python SDK and create, control, and tear down a sandbox from code."
---

Use the Python SDK to drive sandboxes from your own code rather than the CLI.

## Install the SDK

Install the `cua` package into your Python environment.

## Authenticate

Set your Cua API key so the SDK can create cloud sandboxes, or run locally for development.

## Create and control a sandbox

Create a sandbox from an image, run commands and drive the GUI through the sandbox interfaces, then let the sandbox tear itself down.
