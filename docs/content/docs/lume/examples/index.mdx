## Available Examples

### Claude Code

Run the Claude Code CLI in isolated macOS sandboxes using SSH:

- **[Sandbox Setup](/lume/examples/claude-code/sandbox)** — Create and configure a sandboxed macOS VM
- **[Homebrew Testing](/lume/examples/claude-code/homebrew-testing)** — Test Homebrew formulas in a clean environment

### Claude Cowork

Use Claude Cowork with the Lume MCP connector for native VM management:

- **[Sandbox with MCP](/lume/examples/claude-cowork/sandbox)** — Configure the MCP connector for VM lifecycle control
- **[Stock Analysis with Numbers](/lume/examples/claude-cowork/numbers-stock-analysis)** — Use macOS-only apps like Numbers.app
