In this tutorial you create a sandbox, connect the Hermes model to it through Cua Driver, and have it complete a computer-use task end to end.

## What you will build

A working loop where Hermes observes the sandbox screen and issues actions through Cua Driver.

## Prerequisites

The Cua SDK installed, a Cua API key, and access to the Hermes model.

## Steps

1. Create the sandbox from an image.
2. Connect Hermes to the sandbox through Cua Driver.
3. Give it a task and watch it act.
