# Torch-Points3D Overview

**Type:** Torch-Points3D is a **PyTorch framework and library** for deep learning on point clouds, not a standalone GUI application or CLI tool, though it includes training scripts.

## What It Is

As described in the repository: "This is a framework for running common deep learning models for point cloud analysis tasks against classic benchmark. It heavily relies on Pytorch Geometric and Facebook Hydra."

The framework is organized as a Python package (`torch_points3d`) with supporting scripts for training and evaluation.

## Installation

The simplest installation method uses pip:

```bash
pip install torch-points3d
```

For users preferring containerized environments, a Docker image is available:

```bash
docker pull pytorch/pytorch:1.10.0-cuda11.3-cudnn8-devel
```

## Core Requirements

- CUDA 10+ (for GPU support)
- Python 3.7+
- PyTorch 1.8.1+ (1.9+ recommended)
- Optional: Sparse convolution backend

## Primary Use Cases

Torch-Points3D functions as an **importable library** for researchers and developers, offering:
- Pre-built models for various point cloud tasks
- Training via `train.py` script
- Evaluation via `eval.py` script
- High-level API for building custom models

It's not designed as a standalone desktop application but rather as a development framework for point cloud analysis research.
