# Security Policy
## Supported Versions

Only the **current main branch** of this repository is supported with security updates.
All other versions are unsupported.

## Security Issues

Security-related issues must not be reported through public issue trackers or discussions.

## Scope

This policy applies to:

* cua.ai core repositories
* Official CLI, SDKs, and APIs
* Infrastructure and services operated under the cua.ai project

## Cua version
- CUA release version
