Reference documents the full technical specification for Cua's subsystems. It covers `cua-driver`, `sandbox-sdk`, and the `cloud-api`. The pages define commands, APIs, exposed tools, types, and stated limits.

| Section | What it documents |
|---------|-------------------|
| [`cua-driver`](/reference/cua-driver/cli-reference) | CLI commands, 35 MCP tools exposed via the stdio MCP server, and known behavioral limits, including platform-specific constraints on the no-foreground-steal contract. |
| [`sandbox-sdk`](/reference/sandbox-sdk) | Sandbox SDK API, documented by a separate agent. |
| [`cloud-api`](/reference/cloud-api) | CUA Cloud REST API for VM and sandbox management, generated from the OpenAPI specification. |
