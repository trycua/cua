---
title: 'How-to guides'
description: 'Follow recipes for specific Cua goals.'
---

How-to guides are recipes for **a specific goal**. They assume you already know the basics, so do a tutorial first if not. Unlike tutorials, they are *not a lesson*; they get you to a result and stop.

They are grouped by product: Cua Driver for operating apps on a physical machine, Sandbox for cloud desktops, Lume for local Apple silicon VMs, and Cua-Bench for verifiable computer-use tasks. Recipes and agent context cover workflows that span those products.
