In this tutorial you use **Cua Bench** to run Codex and Claude Code against the same computer-use task and compare how they do.

## What you will build

A benchmark run that scores Codex and Claude Code on a task you define.

## Prerequisites

The Cua SDK and Cua Bench installed, a Cua API key, and access to the models you want to compare.

## Steps

1. Define the task and its success criteria.
2. Run the task under Cua Bench for each model.
3. Compare the scores and trajectories.
