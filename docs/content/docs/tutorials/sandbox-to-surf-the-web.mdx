In this tutorial you build a sandbox with a browser and have an agent complete a task on the web, from opening the browser to reading the result.

## What you will build

A sandbox running a browser that an agent drives to complete a web task.

## Prerequisites

The Cua SDK installed and a Cua API key.

## Steps

1. Create a sandbox from an image that includes a browser.
2. Connect an agent to the sandbox.
3. Give it a web task and watch it browse.
