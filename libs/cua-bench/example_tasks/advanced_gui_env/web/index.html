This page is served from a folder using the folder= parameter in launch_window().

How it works:

  • The web/ folder contains this index.html and a style.css file
  • The folder path is passed to env.launch_window(folder="path/to/web")
  • The provider automatically serves the folder contents
  • External resources (CSS, JS, images) are loaded relative to index.html
